User Expierence Design

Going beyond the basics of creating graphics and user interfaces, students focus on how interfaces need to react to user expectations. Students apply previously learned best practices for designing effective interfaces that allow improved work flow while integrating lessons learned by gathering user feedback and usability data. Practical and effective planningfor user interfaces is introduced.

Prerequisites

MTM6161

Learning Materials

Adobe InDesign, InVision, Adobe Illustrator,Adobe XD, Nureva Span

Example Assignments

  1. Wireframes
  2. Paper Prototypes
  3. User Testing

Grading

  1. Assignements 40%
  2. Quizzes 20%
  3. Final Presentation 15%
  4. Final Project 25%

Server Side Web Development

Having static content works well if your website only has a few pages. However, today's modern sites frequently have thousands of pages. Building on the importance of dynamic websites learned through their experiences with Content Management Systems, students leverage databases to store and output content and change websites content and interface elements based on user interactions. The focus is on creating dynamic data and APIs to be used with responsive websites.

Prerequisites

MTM6230

Learning Materials

Brackets, Atom, CodePen, CodeSchool, Mamp

Example Assignments

  1. Contact Book
  2. PHP Blog
  3. HTML to PHP Conversion

Grading

  1. In-Class Exercises 10%
  2. Quizzes 10%
  3. Online Courses 10%
  4. Assignments 20%
  5. Midterm Exam 15%
  6. Final Exam 15%
  7. Group Project 20%
